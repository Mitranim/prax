## Installation

The guide assumes you're using
<a href="https://www.npmjs.com" target="_blank">`npm`</a> and a module-friendly
build system like browserify or Webpack.

Install from `npm`:

```sh
npm i --save-dev prax
```

Then import it in your application:

```javascript
import * as prax from 'prax'
```
